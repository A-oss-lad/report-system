社区规则

为维护江西粤东气象社区的正常秩序，保障社区用户的合法权益，创造一个健康、和谐的网络交流环境，特制定本规范。

一、基本原则

  1. 遵守国家法律法规和社会主义核心价值观。
  2. 尊重他人，友善交流，禁止辱骂、侮辱他人。
  3. 禁止发布色情、暴力、血腥等不良内容。
  4. 禁止发布政治敏感信息和谣言。
  5. 禁止发布与气象无关的广告信息。
  6. 禁止私信骚扰他人。
  7. 保持交流环境的清洁，共同维护良好的社区氛围。

二、违规处理

  1. 对于违反社区规则的行为，管理员有权根据情节轻重采取相应的处理措施。
  2. 轻微违规：警告、禁言1-3天。
  3. 中度违规：禁言7-14天、限制部分功能。
  4. 严重违规：禁言30天、永久禁言、封禁账号。

三、举报机制

  1. 用户发现违规行为可通过举报功能进行举报。
  2. 举报时需提供充分的证据，如截图、聊天记录等。
  3. 管理员将在24小时内对举报进行处理，并反馈处理结果。

四、申诉机制

  1. 用户对处罚有异议的，可在收到处罚通知后7天内提交申诉。
  2. 申诉时需提供充分的证据和理由。
  3. 管理员将在3个工作日内对申诉进行审核，并反馈审核结果。

五、附则

  1. 本规范自发布之日起生效。
  2. 本规范最终解释权归江西粤东气象社区管理团队所有。
  3. 社区管理团队有权根据实际情况对本规范进行修改和完善。

为维护江西粤东气象社区QQ群的正常秩序，保障群成员的合法权益，创造一个健康、和谐的群聊环境，特制定本规范。

一、基本原则

  1. 遵守QQ群管理规定和社区交流规范。
  2. 群聊内容应以气象相关话题为主。
  3. 禁止在群内发布广告、推广等无关信息。
  4. 禁止在群内争吵、辱骂他人。
  5. 尊重群管理员，服从管理。
  6. 保护个人隐私，谨慎分享个人信息。
  7. 鼓励积极参与气象讨论，共同学习进步。

二、群聊礼仪

  1. 入群后请修改群名片，建议使用真实姓名或常用昵称。
  2. 发言前请先查看群公告和历史消息，避免重复提问。
  3. 提问时请简明扼要，提供必要的背景信息。
  4. 回答问题时请耐心、友善，不要嘲笑新手。
  5. 避免刷屏、霸屏等影响他人交流的行为。
  6. 分享文件前请先确认是否有分享价值，并注明文件内容。

三、违规处理

  1. 对于违反群聊规范的行为，群管理员有权根据情节轻重采取相应的处理措施。
  2. 轻微违规：警告、禁言1-3天。
  3. 中度违规：禁言7-14天。
  4. 严重违规：移出群聊、加入黑名单。

四、举报机制

  1. 群成员发现违规行为可通过群举报功能或私聊管理员进行举报。
  2. 举报时需提供充分的证据，如截图、聊天记录等。
  3. 管理员将在24小时内对举报进行处理，并反馈处理结果。

五、附则

  1. 本规范自发布之日起生效。
  2. 本规范最终解释权归江西粤东气象社区QQ群管理团队所有。
  3. 群管理团队有权根据实际情况对本规范进行修改和完善。

常见问题

如何举报违规行为？

您可以通过以下步骤举报违规行为：
1. 登录江西粤东气象社区交流安全中心
2. 点击"我要举报"按钮
3. 填写被举报人信息、违规类型、描述经历等
4. 上传相关证据
5. 提交举报

如何查看举报处理进度？

您可以通过以下步骤查看举报处理进度：
1. 登录江西粤东气象社区交流安全中心
2. 点击"处理进度"按钮
3. 在列表中查看您提交的举报及其处理状态
4. 点击"查看详情"可查看更详细的处理信息

如何对处罚进行申诉？

您可以通过以下步骤对处罚进行申诉：
1. 登录江西粤东气象社区交流安全中心
2. 点击"违规处罚通知"按钮
3. 在列表中找到您要申诉的处罚通知
4. 点击"申诉"按钮
5. 填写申诉理由并上传相关证据
6. 提交申诉

举报处理需要多长时间？

我们会在收到举报后的24小时内进行初步审核，并在3个工作日内给出最终处理结果。 对于复杂的举报案件，处理时间可能会延长，我们会尽快处理并通知您。

申诉处理需要多长时间？

我们会在收到申诉后的3个工作日内进行审核，并给出最终处理结果。 对于复杂的申诉案件，处理时间可能会延长，我们会尽快处理并通知您。

关于我们

江西粤东气象社区成立于2015年，是一个专注于气象知识交流、气象信息分享的社区平台。 我们致力于为广大气象爱好者提供一个健康、和谐的交流环境，促进气象知识的传播和普及。

社区目前拥有江西气象爱好者和粤东气象爱好者两个主要QQ群，以及多个专题讨论组。 社区成员包括气象专业人士、气象爱好者、学生等，大家共同探讨气象知识，分享气象信息。

为了维护社区的正常秩序，保障社区用户的合法权益，我们成立了江西粤东气象社区交流安全中心， 负责处理社区内的违规行为举报、处罚申诉等事务，确保社区交流环境的安全和规范。
