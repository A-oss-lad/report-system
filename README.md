# 江西粤东气象社区交流安全中心系统

## 项目简介

江西粤东气象社区交流安全中心系统是一个专为气象社区设计的安全管理平台，主要用于社区用户的登录注册、举报违规行为、查看处理进度、接收违规处罚通知及申诉，同时支持管理员对举报、申诉、处罚等进行后台管理，保障社区交流环境安全规范。

## 功能特点

### 普通用户功能
- **登录/注册**：支持QQ号码和密码登录
- **举报功能**：可提交举报表单，包含被举报人信息、违规类型、描述经历、附件证据等
- **处理进度**：查看已提交举报的处理状态和详情
- **违规处罚通知**：接收违规处罚通知，查看详情并进行申诉
- **申诉功能**：对处罚结果进行申诉，提交申诉理由和证据
- **个人中心**：查看个人信息、统计数据和最近活动

### 管理员功能
- **控制台**：查看关键数据统计和图表
- **用户管理**：管理社区用户账号
- **团队管理**：管理社区管理团队
- **举报处理**：处理用户提交的举报，查看详情并做出处理决定
- **申诉处理**：处理用户提交的申诉，查看详情并做出处理决定
- **处罚管理**：对违规用户进行处罚，设置处罚类型和时长
- **规范管理**：维护社区规则和规范
- **公告管理**：发布社区公告
- **系统设置**：配置系统参数

## 数据存储

本系统使用浏览器的 localStorage 和 sessionStorage 进行数据存储，包括：

- 用户信息
- 举报记录
- 申诉记录
- 处罚记录
- 规则和公告
- 系统设置

## 注意事项

1. 本系统为前端演示版本，实际应用中需要后端支持
2. 数据存储在浏览器本地，清除浏览器数据会导致数据丢失
3. 附件上传功能仅为前端演示，实际应用中需要后端支持文件存储
4. 系统中的时间和日期显示基于用户本地时间
